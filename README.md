# Library Inventory Manager
This is my mini project for Python.
A command-line mini project to manage books in a campus library.

## Features
- Add, issue, return and search books
- JSON file storage for persistence
- Object-oriented design with Book and LibraryInventory classes
- Exception handling and logging

## How to Run
1. Open a terminal in the project root: `library-inventory-manager-yourname`
2. Run: `python -m cli.main`
3. Use the menu to manage the library(to add, issue, return and search books.).
